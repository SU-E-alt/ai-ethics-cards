AI Bootcamp Human-in-the-Loop Ethics Card Deck
This project provides an interactive card deck for exploring ethical accountability and context-aware use of AI systems. Each table receives a unique seeded shuffle to guide discussion, supporting UNESCOs Human-in-the-Loop AI principles.
Features

Seeded shuffle per table using URL parameters
Draw one or two cards to combine prompts
Ethical challenges and reflective prompts grounded in real-world AI contexts
Summary copy and submission options
Image support for each card
Setup

Clone or download the repository.
Open index.html in a browser.
Use the buttons to draw cards and explore ethical prompts.
Customize the seed in script.js for different tables.
Customization

Add more cards to the CARDS array in script.js.
Include images by adding an image field to each card.
Replace the placeholder form URL in script.js with your actual Google Form link.
